## Important constraint

The live Motion theme and the current Dawn-based theme do not share the same
section architecture. Most Motion sections cannot be copied directly. The safe
approach is:

1. Keep Motion as a content reference
2. Reuse compatible global settings
3. Remap live copy and merchandising intent into Dawn sections
4. Rebuild complex Motion-specific layouts only when they are still worth the cost
