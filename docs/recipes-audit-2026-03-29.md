# Audit recettes - 29 mars 2026

## Constat

Le socle produit est solide :

- experience de lecture tres au-dessus d'une fiche Shopify standard
- navigation recette claire
- recherche, filtres, liste de courses et liens catalogue deja utiles
- back-office recette assez avance pour produire vite

Le point faible principal n'est plus technique. Il est editorial :

- plusieurs textes front sonnaient encore "studio", "prototype" ou "outil"
- certaines fiches avaient des sous-titres trop conceptuels au lieu d'etre gourmands et concrets
- des accents ou tournures trop "ASCII" faisaient baisser la perception premium
- plusieurs cartes hub restent faibles tant que les vrais visuels ne sont pas poses

## Passe corrigee

Cette passe a nettoye :

- le wording du hub recettes
- les placeholders et messages systeme visibles
- les labels de collections trop internes
- plusieurs resumes, sous-titres et descriptions SEO visibles sur les fiches

## Priorites par bloc

### 1. Hub recettes

A continuer :

- verifier chaque panneau collection
- remplacer tous les visuels vides
- resserrer encore certaines phrases de cartes si la ligne casse visuellement

### 2. Fiches recette

A continuer :

- harmoniser les rubriques `Pourquoi`, `Erreurs a eviter`, `Service`
- corriger tous les accents restants dans les contenus de fiche
- verifier que chaque recette a bien une promesse gustative tres concrete

### 3. Repertoire

A continuer :

- monter de 15 a 40 recettes
- construire des familles editoriales fortes :
  - desserts a la vanille
  - petit-dejeuner et gouter
  - vanille salee
  - recettes rapides
  - recettes signature

### 4. Medias

A continuer :

- hero sur toutes les recettes
- galerie sur les recettes piliers
- medias d'etapes sur les recettes les plus importantes
