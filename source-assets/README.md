# Source Assets

Ce dossier contient les medias bruts prepares avant l'integration dans le theme Shopify.

## Structure

- `branding/` : elements d'identite visuelle, logos
- `images/` : photos sources a recadrer, optimiser ou renommer avant integration
- `videos/` : videos sources a convertir ou compresser avant integration

## Inventaire actuel

- `branding/` : 1 fichier
- `images/` : 31 fichiers
- `videos/` : 9 fichiers

## Regles pour la suite

- Ne pas utiliser ces fichiers bruts directement comme `assets` Shopify.
- Conserver les noms d'origine tant que la selection editoriale n'est pas faite.
- Creer ensuite des fichiers web-optimises et nommes proprement pour le theme.
- Les noms avec suffixe ` 2` ont ete conserves tels quels pour eviter toute perte de correspondance.
